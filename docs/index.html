Memory Savings内存节省

27B-4bit: 14.09 GB → 1.70 GB (88% reduction). 9B-8bit: 8.86 GB → 2.29 GB (74%). Both fit in 8GB with streaming. 27B-4bit: 14.09 GB → 1.70 GB（减少 88%）。9B-8bit: 8.86 GB → 2.29 GB（74%）。两者通过流式均可在 8GB 设备运行。

I/O BottleneckI/O 瓶颈

Streaming is I/O-bound: compute/load ratio = 1:1660 on iPad. Layer strategy is irrelevant — only resident count matters. 流式是 I/O 瓶颈：计算/加载比 = 1:1660（iPad）。层策略无关紧要 — 只有驻留数量重要。
